Intro to CSS

Learning about some basic CSS properties

During our last class, we learned about the basic CSS grammar: selectors, properties and values. During this class, we will practice this grammar, learn some more basic properties, and learn one of the three ways to include style information in an HTML document.

References

During this exercise, we'll refer to the CSS reference provided by the Mozilla Developers Network (MDN). You may want to bookmark this for future exercises. It is a good idea to have a reference handy to double check small points of syntax, especially the values permitted for a given property.
